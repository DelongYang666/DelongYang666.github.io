How to Blog With Markdown

October 7, 2024

This blog is suitabe for someone who want to blog with markdown.It introduces some common grammar of markdown which is applicable to blogging.If you are familiar with the website technology such as html,css and js, it is easy for you to understand the content of the blog.

top

outline

  • 1 how to write words
  • 2 how to add code blocks
  • 3 how to add links from outside
  • 4 how to do the title jump to the paragraph in the page
  • 5 how to go back the top from the bottom of the page
  • 6 how to use escape character

1 how to write words

big title press

# big title

medium title press

## medium title

small title press

### small title

# is similar to h1,## is similar to h2,### is similar to h3 in html.

2 how to add code blocks

single line code press

'singgle line code'

multi line code press

'’' multi line code multi line code multi line code '''

3 how to add links from outside

[Delongyang666’s blog](url)<=>Delongyang666’s blog

url is the website url from outside.

4 how to do the title jump to the paragraph in the page

title press

[title](#paragraph)

paragraph press

# paragraph

the paragraph should be the same as the content behind #.If there are a few words in the paragraph, you should use ‘-’ to connect them after the #.For example, if the content in the paragraph is ‘how to do jump’, you should write ‘how-to-do-jump’ behind the # in the title.Any other symbols are not allowed maybe(I’m not sure,but I am not succesful when I use ‘:’).You can use numbers or characters.

5 how to go back the top from the bottom of the page

the theory is the same as the previous one(4 how to do the title jupm to the paragraph in the page)

on the top of the page press

# top

on the bottom of the page press

[back to the top](#top)

6 how to use escape character

you just need add \ in front of the character you want to use.For example, if you want to use #, you just need add \ in front of # like \#.
