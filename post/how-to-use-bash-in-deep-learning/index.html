How to Use Bash in Deep Learning

October 8, 2024

This blog is suitable for someone who want to use bash to train in deep learning. It introduce some common usage of bash.

1 how to run bash script

to run a bash script, you should add #!/bin/bash in the first line of the script.

‘#!/bin/bash’

It tells the system that this script should be run in bash.

’’' sh script.sh ’''

If you want to pass three parameters to the script, you can use p1 p2 p3 behind the command to get the parameters.

’’' sh script.sh p1 p2 p3 ’''

In bash script, you can use variables to store the parameters and pass to the script.

’’' model_name = Qwen2.5 sh script.sh $model_name …

2 how to run python script in bash script

’’' python script.py p1 p2 p3 ’''

3 how to detect the error and stop immediately

’’' set -e ’''

4 how to print parameters to debug when running bash script

’’' model_name = Qwen2.5 echo $model_name ’''

5 how to use deepspped to start training in bash script

’’' deepspeed –num_gpus=8 train.py –parameter1 parameter1 –parameter2 parameter2 ’''
